第一步
关键是 ！不想准备pkl文件！


第1步、ESMFold安装好，用法：
conda activate esmfold
python scripts/fold.py -i example/test_protein.fa -o example/pdb --cpu-offload
如果序列大于850 ，要加--chunksize 128


第2步 提取原始的语言模型特征
python scripts/ProtTrans.py --fasta example/test_protein.fa --out_path example/rawembedd --gpu 0

第三步  根据train 的最大最小值  归一化语言模型特征

python scripts/process_ProtTrans.py



第四步: 从pdb 提取dssp特征
python scripts/get_dssp.py

第五步：合并特征
python scripts/pad_feature.py


第六步  预测


CUDA_VISIBLE_DEVICES=0 python ./infer.py --task PRO --test --seed 2022 --run_id weights --dataset_path ./infer_data/PRO/ --feature_path ./infer_data/PRO/
